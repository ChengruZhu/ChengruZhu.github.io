发表于 2018-08-07 | 分类于 ubuntu

问题1：重新安装ubuntu16.04后,ifconfig 会发现网卡名已经由 udev 管理命名为 eth0 变成了enp4s0f1 wlan0变成了wlp3s0 或者类似的名称.

解决方法：

找到GRUB_CMDLINE_LINUX=””改为GRUB_CMDLINE_LINUX=”net.ifnames=0 biosdevname=0”
然后

重启后，网卡名称果然变成了eth0和wlan0

##问题3：设置代理ubuntu16.04 + shadowsocks-qt5 + genpac
安装 shadowsocks-qt5

安装genpac

使用genpac生成autoproxy.pac

然后在你的/home/xxx/下面会生成一个autoproxy.pac
然后开始使用，在搜索栏里搜索shadowsocks-qt5运行它，配置它，无非是填写你的服务器地址密码加密方式等，登录后注意看延时，假如有延时数据证明你连接上了，否则你服务器有问题
